# Automatic-Attendence-System-with-Facial-Recognition

While using this you can use Python version 3.10 to 3.8 so that the facial recognition library doesn't create any error.
Another error that you can encounter is a dlib error or building wheels; you may need to externally download that zip file. There are many tutorials available on YouTube for the solution.
Also make sure to install all libraries through the terminal for the same.
That's all, you are good to go.

To run this on your device just write `pip install -r requirements.txt` in your terminal.

I have also provided a link for the dlib error if it occurs:
